# All Leagues

## Endpoint

This is the endpoint you will be using to get information of all leagues listed.

```{1} filename="Endpoint" copy
https://football-api67.p.rapidapi.com/allLeagues
```

## How to use this API

### Request Headers

Set the request headers according to the listed headers, unless you will get an error response from the API.

```js filename="Request Headers" copy
{
  method: "GET"
  headers: {
    'X-RapidAPI-Key'  : YOUR_API_KEY,
    'X-RapidAPI-Host' : 'football-api67.p.rapidapi.com',
  }
}
```

### Query Parameters

This endpoint is **not require** any parameter, just go with this endpoint and you will get response from the API.

```{1} filename="Parameters" copy
https://football-api67.p.rapidapi.com/allLeagues
```

### Example Request

This is the example request using [`Node.js`](https://nodejs.org/) fetch.

```js filename="example.js" copy
const fetch = require("node-fetch");

const url = "https://football-api67.p.rapidapi.com/allLeagues";
const options = {
  method: "GET",
  headers: {
    "X-RapidAPI-Key": YOUR_API_KEY,
    "X-RapidAPI-Host": "football-api67.p.rapidapi.com",
  },
};

try {
  const response = await fetch(url, options);
  const result = await response.text();
  console.log(result);
} catch (error) {
  console.error(error);
}
```
