# League Info

## Endpoint

This is the endpoint you will be using to get information such as league name, id, and list of seasons available.

```{1} filename="Endpoint" copy
https://football-api67.p.rapidapi.com/leagues
```

## How to use this API

### Request Headers

Set the request headers according to the listed headers, unless you will get an error response from the API.

```js filename="Request Headers" copy
{
  method: "GET"
  headers: {
    'X-RapidAPI-Key'  : YOUR_API_KEY,
    'X-RapidAPI-Host' : 'football-api67.p.rapidapi.com',
  }
}
```

### Query Parameters

This endpoint also requires a parameter. `League ID` is required to get the league information from the API.
The final api endpoint would be like this as you can see the example requests below.

```{1} filename="Parameters" /LeagueID/ copy
https://football-api67.p.rapidapi.com/leagues?id=LeagueID
```

### Example Request

This is the example request using [`Node.js`](https://nodejs.org/) fetch.

```js filename="example.js" copy
const fetch = require("node-fetch");

const url = "https://football-api67.p.rapidapi.com/leagues?id=42";
const options = {
  method: "GET",
  headers: {
    "X-RapidAPI-Key": YOUR_API_KEY,
    "X-RapidAPI-Host": "football-api67.p.rapidapi.com",
  },
};

try {
  const response = await fetch(url, options);
  const result = await response.text();
  console.log(result);
} catch (error) {
  console.error(error);
}
```
