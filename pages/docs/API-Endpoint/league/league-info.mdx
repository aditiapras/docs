# League Info

## Base URL

This is the base URL for this endpoint `https://football-api67.p.rapidapi.com/leagues` you can use this endpoint to get the league information such as league names, id, and seasons.

### Request Headers

Dont forget to set the request headers according to the listed headers, unless you will get an error response from the API.

`'X-RapidAPI-Key': YOUR_API_KEY` and `'X-RapidAPI-Host': 'football-api67.p.rapidapi.com'`

### Query Parameters

This endpoint also requires a parameter. `League ID` is required to get the league information from the API.
The final api endpoint would be like this `https://football-api67.p.rapidapi.com/leagues/?id=LeagueID`. You can see the example requests below.

### Example Request

This is the example request using [`Node.js`](https://nodejs.org/) fetch.

```js filename="example.js" copy
const fetch = require("node-fetch");

const url = "https://football-api67.p.rapidapi.com/leagues?id=42";
const options = {
  method: "GET",
  headers: {
    "X-RapidAPI-Key": YOUR_API_KEY,
    "X-RapidAPI-Host": "football-api67.p.rapidapi.com",
  },
};

try {
  const response = await fetch(url, options);
  const result = await response.text();
  console.log(result);
} catch (error) {
  console.error(error);
}
```
