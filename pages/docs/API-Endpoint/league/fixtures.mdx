## Endpoint

This endpoint you will give information about the new fixtures in the league.

```{1} filename="Endpoint" copy /leagues/ /fixtures?id=42/
https://football-api67.p.rapidapi.com/leagues/fixtures?id=42
```

## How to use this API

### Request Headers

Set the request headers according to the listed headers, unless you will get an error response from the API.

```js filename="Request Headers" copy
{
  method: "GET"
  headers: {
    'X-RapidAPI-Key'  : YOUR_API_KEY,
    'X-RapidAPI-Host' : 'football-api67.p.rapidapi.com',
  }
}
```

### Query Parameters

This endpoint is requires `League ID` as paramaters to get data about league fixtures from the API endpoint.

```{1} filename="Parameters" copy /id=42/
https://football-api67.p.rapidapi.com/leagues/fixtures?id=42
```

### Example Request

This is the example request using [`Node.js`](https://nodejs.org/) fetch.

```js filename="example.js" copy
const fetch = require("node-fetch");

const url = "https://football-api67.p.rapidapi.com/allLeagues";
const options = {
  method: "GET",
  headers: {
    "X-RapidAPI-Key": YOUR_API_KEY,
    "X-RapidAPI-Host": "football-api67.p.rapidapi.com",
  },
};

try {
  const response = await fetch(url, options);
  const result = await response.text();
  console.log(result);
} catch (error) {
  console.error(error);
}
```
