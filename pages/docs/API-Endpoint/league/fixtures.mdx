## Base URL

This is an example of component using data from the API. This data is available in real-time, once the match is finished it will gather new fixtures for the next match. From this endpoint you will get information about the new fixtures in the league.

Base URL: `https://football-api67.p.rapidapi.com/leagues/fixtures`

### Request Headers

Dont forget to set the request headers according to the listed headers, unless you will get an error response from the API.

`'X-RapidAPI-Key': YOUR_API_KEY` and `'X-RapidAPI-Host': 'football-api67.p.rapidapi.com'`

### Query Parameters

This endpoint require 2 (two) query parameters.
